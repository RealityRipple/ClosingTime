# ClosingTime
Hide the Close button on tabs that have a BeforeUnload prompt pending.

#### Supports
 * Pale Moon [34.0 - 34.*]

## Building
Simply download the contents of the repository and pack the contents (sans git data) into a .zip file. Then, rename the file to .xpi and drag into the browser.

## Download
You can grab the latest release from the [Official Web Site](//realityripple.com/Software/XUL/ClosingTime/).
